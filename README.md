# EEG Psychiatric Disorder Classification

## Requirements

- Python 3.12

## Setup

```bash
# Install dependencies
pip install -r backend/requirements.txt
```

## Run

```bash
# Terminal 1 - Backend
cd backend
python main.py

# Terminal 2 - Frontend
cd frontend
python3 -m http.server 3000
```

Open http://localhost:3000
